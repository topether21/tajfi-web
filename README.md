## Get started

1. Install dependencies

   ```bash
   bun i
   ```

2. Start the app

   ```bash
    bun run start
   ```

## (Deploy Web)[https://docs.expo.dev/distribution/publishing-websites/]

```
bun i
bun run build:web
bun run serve:web
```

## gluestack

bunx gluestack-ui init --use-bun
